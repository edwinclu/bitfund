#Notes to use IB web api
1. Download & Install java runtime: https://www.java.com/en/download/
2. Download & Run IB Client Portal Gateway: https://www.interactivebrokers.com/campus/ibkr-api-page/cpapi-v1/#run-gw
3. Sign in
4. Optional: Bypass or Generate Certificate: https://github.com/Voyz/ibeam/wiki/TLS-Certificates-and-HTTPS
5. Trader doc: https://interactivebrokers.github.io/cpwebapi/

#Python
1. create python environment: conda create -n trade python=3.11 (once)
2. conda activate trade (everytime)
3. 'poetry add xx' to add a lib
4. 'poetry install' after git clone
5. python src/test.py
